# Changelog

All notable changes to Medena Care are documented in this file.

---

## February 2026

### Added
- **Documentation System**: Comprehensive documentation with bilingual user manuals and technical reference
- **Prescription Pricing Management**: Admin can now set prescription prices globally and per doctor
- **ICD-10 for Prescriptions**: Doctors must provide ICD-10 code when completing prescription requests

### Fixed
- ICD-10 input field now visible for prescription request completions

---

## January 2026

### Added
- **Prescription Request Flow**: Patients can request prescription renewals without full consultation
- **Consultation Reports**: Downloadable PDF reports for completed consultations
- **Honorarnote Generation**: Auto-generated medical fee notes for insurance reimbursement
- **Doctor Referral System**: QR codes, embed widgets, and referral links for B2B doctors
- **Doctor Public Profiles**: Safe public data for referral lookups without exposing PII

### Improved
- Patient dashboard shows assigned doctor with avatar
- Enhanced consultation flow with better mobile experience
- Prescription requests marked with Rx badge in doctor dashboard

---

## December 2025

### Added
- **Doctor Dashboard**: Consultation queue with tabs for new, in-progress, and completed cases
- **Admin Dashboard**: Platform statistics, doctor management, pricing configuration
- **Multi-role Authentication**: Separate login flows for patients, doctors, and admins
- **Stripe Integration**: Secure payment processing with customizable pricing

### Improved
- Body location picker with front/back views
- Photo upload with guidelines and validation
- Symptom severity slider

---

## November 2025

### Initial Release
- **Core Platform**: 10-step consultation flow
- **Patient Dashboard**: View active and past consultations
- **Basic Authentication**: Patient signup and login
- **Bilingual Support**: English and German languages

---

*This changelog is maintained alongside feature development. Each entry includes the date, summary, and affected areas.*
